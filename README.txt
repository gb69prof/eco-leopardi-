# L'eco di Leopardi – prototipo (ZIP)

## Struttura
- `index.html` (home, unica pagina nel root)
- `storico/storico.html` (sezione Pessimismo storico)
- `storico/sezioni/*` (placeholder collegati)
- `assets/` (css, js, img)

## Come provarlo subito
1. Apri `index.html` con un server locale (consigliato).
   - Se hai Python: `python3 -m http.server 8000` nella cartella del progetto.
2. Vai su `http://localhost:8000`

## Comportamenti
- Click sullo **schema** nel testo → si apre nel frame sinistro.
- Footer: **Saperi / Vocabolario / Test** → si aprono nel frame sinistro (con pulsante "Torna al video").

## Nota
Le immagini `home-bg.jpg` e `leopardi-portrait.jpg` sono placeholder: puoi sostituirle con i tuoi asset.
